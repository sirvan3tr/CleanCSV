# CleanCSV
Strip and replace unwanted characters from your CSV cells.

Using Python 3.9

### Usage
- ```python clean_csv.py [INPUT FILE] [OUTPUT FILE]```
- ```python clean.csv.py example.csv output_cleaned.csv```
